  1. gdb - debugger for C. 
        syntax intel set: $echo "set disassembly-flavor intel" > ~/.gdbinit
    -q - ?
    -g - bonus information for gdb
    Interaction: 
        (gdb) break main
        (gdb) run
        (gdb) info regusters
        (gdb) info reguster <register> | i r <register>
        (gdb) quit
        (gdb) list
        (gdb) disassembly main
        (gbd) print
        (gbd) nexti
        (gbd) runt test
        (gbd) where
    Examine:
        (gdb) x/o 0x5555....
        (gdb) x/x $rip | x/x, x/o, x/u, x/t
        (gdb) x/12o $rsi 
        (gdb) x/i $rip | instruction
        (gdb) x/c $rip | char
        (gdb) x/s $rip | string
    bt: backtrace. Examine stack. Every record in bt listing is stack frame. 
        (gdb) bt
        (gdb) bt full | local variables
    
    (gdb) operation <destination>, <source>
