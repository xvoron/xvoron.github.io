Variables

            void func(){
      static int static_var = 5;
      static_var ++;
      printf("%d\n",static_var)
      }
      
      int main(){
      func();
      func();
      return 0;
      }
      print:
      > 5
      > 6

File access

  • descriptors: fd. If there is a error return -1
    • open() - return int
    • close()
    • read()
    • write() - need to know how many bytes will write

In #include add: fcntl.h, sys/stat.h ---> flags to open() function

flag description
O_RDONLY Read only
O_WRONLY Write only
O_RDWR Read and write
-------- -------------
use with OR
------- ------------
O_APPEND Append date to the end
O_TRUNC If file exist, change len to 0
O_CREAT If file do not exist, create a file
  • filestreams

File permissions

Add in #include sys/stat.h. Can be combined with OR operator.

flag description
S_IRUSR Read owner
S_IWUSR Write owner
S_IXUSR Execute owner
S_IRGRP Read groupe
S_IWGRP Write groupe
S_IXGRP Execute groupe
S_IROTH Read other
S_IWOTH Write other
S_IXOTH Execute other
