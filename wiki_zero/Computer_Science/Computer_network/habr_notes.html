Basics

What is the Computer network?

A computer network is a digital telecommunications network for sharing resources between nodes, which are computing devices that use a common telecommunications technology.

Components of Computer network:

  • Endpoint devices (Hosts, Communication endpoint, end systems/stations).
  • Intermediary devices (Reapeaters, hubs, Bridges, Switches, Routers, Modems, Firewalls).
  • Media (Copper media, Optical meda, Wirless media)

Service provided by Computer network:

  • Apllication
    • Downloaders (FTP, TFTP)
    • Mails
    • Messangers
    • Real-time applications (Skype, Viber)
  • Network resources (Printers, camers)
  • Storage
  • Backups
  • VoIP

Topology:

  • Physical - where nodes will be?
    • Bus topolohy.
    • Ring topology.
    • Star topology.
    • Full-Mesh topology.
    • Partial-Mesh topology.
    • Hybrid topology - topology of nowadays internet
  • Logical - which nodes packege will use?

Network Models:

  • OSI
    1. Application layer - applications, mail, web-browsers, HTTP, FTP.
    2. Presentation layer - retranslate information, ASCII.
    3. Session layer - connecting 2 hosts, closing connection.
    4. Transort layer - transport, ports, TCP.
    5. Network layer - roating. IPv4, IPv6.
    6. Data link layer - addressing, check data. MAC, Ethernet.
    7. Physical layer - electric signals.
  • TCP/IP (DoD - Department of Defence)
    1. Application layer
    2. Transport layer
    3. Internet layer
    4. Network interface

ARP

ARP (Address Resolution Protocol) wiki

ARP-request -> sends to ff:ff:ff:ff:ff:ff and said: "Where are you somebody with this IP? If there any please answer me." ARP-answer -> "Yes Im here with this MAC-address."

There is a ARP-chashe, where the data MAC/IP has been storage.

hardware type protocol type
hardware length protocol length operation
sender hardware address
sender protocol address
target hardware address
target protocol address
  1. Harware type - data link level protocol: Ethernet.
  2. Protocol type - network level protocol: IPv4, IPv6.
  3. Hardware length - MAC-address: 6 byte.
  4. Protocol length - IPv4, IPv6 address.
  5. Operation - request = 0x0001, responde = 0x0002.
  6. Sender hardware address - MAC-address sender.
  7. Sender protocol address - IP-address sender.
  8. Target hardware address - MAC-address target. If request = empty.
  9. Target protocol address - IP-address target.
