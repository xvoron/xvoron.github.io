VIMWIKI

:VimwikiSearch /pattern/ :VWS /pattern/

Search for /pattern/ in all files of current wiki. To display all matches use |:lopen| command. To display next match use |:lnext| command. To display previous match use |:lprevious| command.
Hint: this feature is simply a wrapper around |:lvimgrep|. For a description of how the pattern can look like, see |:vimgrep|. For example, to do a case insensitive search, use > :VWS /\cpattern/
