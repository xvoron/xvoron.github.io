UNIX File permissions

file typer u u u g g g o o o
- r w x r w x r - -

u - user, g - group, o - other, s - setuid (set user id). If there is a s parametr this command will execute with owner of file permisiions.

chmod usage: 

chmod 777 file.c
chmod [ugo][+/-][rwx] file.c
